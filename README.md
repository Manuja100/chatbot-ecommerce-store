# chatbot-ecommerce-store

<h3>Chatbot for an E-commerce store selling clothes</h3>

Features -

<ul>
  <li>Respond to users</li>
  <li>Give users product info</li>
  <li>Track orders of users when the order ID is given</li>
</ul>

Technologies used

<ul>
  <li>Python</li>
  <li>Tensorflow</li>
  <li>Flask</li>
  <li>nltk</li>
</ul>
